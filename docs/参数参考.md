# 参数参考

本文记录工程中已经纳入配置对象管理的默认参数。当前阶段只建立默认配置与说明，不改变算法行为；后续再逐步支持 TOML/CLI 覆盖和参数校验。

## 通用 RadioConfig

代码位置：`common/config.py`

| 参数 | 默认值 | 单位 | 用途 | 调参影响 |
|------|--------|------|------|----------|
| `target_sample_rate_hz` | 48000 | Hz | 离线窄带解码目标采样率 | 改动会影响所有符号采样点数和同步定位假设，需整体联动 |
| `wideband_sample_rate_hz` | 2500000 | Hz | 未能从文件名识别采样率时的宽带默认采样率 | 只应匹配实际 IQ 文件采样率 |
| `wideband_resample_up` | 12 | ratio | 2.5 MHz 到 48 kHz 的有理重采样上采样因子 | 与 `wideband_resample_down` 成对使用 |
| `wideband_resample_down` | 625 | ratio | 2.5 MHz 到 48 kHz 的有理重采样下采样因子 | 与 `wideband_resample_up` 成对使用 |
| `narrowband_max_sample_rate_hz` | 200000 | Hz | `scanner.py` 判断窄带/宽带路径的采样率分界 | 调低会让更多文件走宽带候选搜索，调高会让更多文件直接窄带处理 |
| `psd_peak_threshold_db` | 15 | dB | 宽带 PSD 候选检测门限，高于中值噪声底的 dB 数 | 调高减少误检但可能漏检弱信号，调低增加候选和计算量 |
| `psd_nperseg` | 4096 | samples | Welch PSD 分段长度 | 增大提升频率分辨率但需要更长数据和更多计算 |
| `psd_peak_min_distance_bins` | 20 | FFT bins | PSD 峰值之间的最小间隔 | 调大减少重复候选，调小可能重复检测同一信号 |

## DMRConfig

代码位置：`dmr/config.py`

| 参数 | 默认值 | 单位 | 用途 | 调参影响 |
|------|--------|------|------|----------|
| `target_sample_rate_hz` | 48000 | Hz | DMR 解码目标采样率 | 影响 DMR `SPS=10` 假设 |
| `wideband_sample_rate_hz` | 2500000 | Hz | DMR 历史宽带默认采样率 | 应与输入文件采样率一致 |
| `symbol_rate_hz` | 4800 | symbols/s | DMR 符号率 | 协议固定值，通常不调整 |
| `samples_per_symbol` | 10 | samples/symbol | 当前 48 kHz 下的 DMR 每符号采样数 | 与目标采样率和符号率联动 |
| `frontend_cutoff_hz` | 9500 | Hz | DMR/P25 共用 FSK 前端低通截止 | 调低可能削弱信号边带，调高可能引入邻道噪声 |
| `frontend_taps` | 151 | taps | 前端 FIR tap 数 | 增大提高滤波器选择性但增加计算量 |
| `nominal_deviation_hz` | 1944 | Hz | 4FSK 幅度归一化参考频偏 | 影响四电平判决尺度 |
| `sync_threshold_voice` | 0.68 | NCC | 语音同步相关门限 | 调高减少误检但可能漏检低质量语音同步 |
| `sync_threshold_data` | 0.55 | NCC | 数据同步相关门限 | 调高减少误检但可能漏检控制突发 |
| `voice_burst_stride_samples` | 2880 | samples | 同一 slot 语音突发间隔 | 与 60 ms TDMA 帧和 48 kHz 采样率绑定 |

## DPMRConfig

代码位置：`dpmr/config.py`

| 参数 | 默认值 | 单位 | 用途 | 调参影响 |
|------|--------|------|------|----------|
| `target_sample_rate_hz` | 48000 | Hz | dPMR 解码目标采样率 | 影响 `SPS=20` 假设 |
| `symbol_rate_hz` | 2400 | symbols/s | dPMR 符号率 | 协议固定值，通常不调整 |
| `samples_per_symbol` | 20 | samples/symbol | 当前每符号采样数 | 与目标采样率和符号率联动 |
| `frontend_cutoff_hz` | 3500 | Hz | dPMR 专用窄带前端低通截止 | 调高可能引入邻道噪声，调低可能削弱有效调制成分 |
| `frontend_taps` | 151 | taps | 前端 FIR tap 数 | 增大提高滤波器选择性但增加计算量 |
| `nominal_deviation_hz` | 1944 | Hz | 四电平归一化参考频偏 | 影响符号判决尺度 |
| `sync_threshold` | 0.82 | NCC | dPMR FS1/FS2 同步检测门限 | 调高减少误检但可能漏检低质量帧 |
| `sps_search_min` | 19.0 | samples/symbol | dPMR 符号恢复 SPS 搜索下界 | 扩大范围提高容错但增加计算量和误候选 |
| `sps_search_max` | 21.0 | samples/symbol | dPMR 符号恢复 SPS 搜索上界 | 扩大范围提高容错但增加计算量和误候选 |
| `sps_search_steps` | 21 | steps | dPMR SPS 搜索步数 | 增大提高搜索精度但增加计算量 |
| `frame_symbols` | 384 | symbols | dPMR 基本帧长度 | 协议结构参数，通常不调整 |

## RealtimeConfig

代码位置：`common/config.py`

| 参数 | 默认值 | 单位 | 用途 | 调参影响 |
|------|--------|------|------|----------|
| `active_threshold_db` | 15 | dB | realtime detector 活跃信道能量门限 | 调高减少误检但可能漏检弱信号 |
| `channel_grid_hz` | 12500 | Hz | detector 频率量化栅格 | 应与目标信道间隔匹配 |
| `close_hysteresis_windows` | 3 | windows | 连续静默多少窗口后关闭信道 | 调大更稳但呼叫关闭变慢 |
| `call_timeout_windows` | 5 | windows | 聚合器超时关闭呼叫的窗口数 | 调大可容忍更长间断但延迟关闭 |
| `fo_bucket_hz` | 5000 | Hz | 呼叫/去重频偏桶宽度 | 调大更容易合并近邻候选，调小更容易拆成多个候选 |
